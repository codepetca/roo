Student Auth Debug Test

This page helps debug the student authentication form.

Instructions

  1. Click "Open Login Page"
  2. Select "Student" from the role selection
  3. Click "Create student account" at the bottom
  4. Look at the debug panel to see current state values
  5. Type in the email and password fields
  6. Check if the debug panel updates and if the button becomes enabled

Debug Console

Open browser developer tools to see console output from the StudentAuth component.
